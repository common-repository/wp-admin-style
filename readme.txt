== Description ==
Adds styles to wp-admin pages. Most popular Black with Pink style included.

== Installation ==
1. Upload `wp-admin-style` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
Can I disable Black with Pink style? =
Yes, just chose Default style.
